## Projects

### Retail Sales Analysis Dashboard  
**Tools:** Power BI, DAX, Power Query  
🔗 [View Project](https://github.com/nguyenngochonghoa/Power-BI-Retail-Sales-Analysis-Dashboard)

This project presents an interactive Power BI dashboard to analyze retail sales performance and support business decision-making. A star-schema data model was built to track key KPIs such as revenue, units sold, and average order value across time, regions, channels, and product segments.
The analysis reveals revenue concentration in the East region, strong performance of Online and Outlet channels, and a high proportion of unsold products. At the product level, high-performing segments (Productivity, Extreme) contrast with underperforming SKUs, highlighting opportunities for portfolio optimization.
Based on these insights, recommendations were proposed to improve channel strategy, reduce inventory inefficiencies, and focus on high-value segments. This project demonstrates skills in data modeling, Power BI, DAX, and translating data into actionable business insights.

### US Superstore Business Analytics  
**Tools:** Excel, Power Query  
🔗 [View Project](https://github.com/nguyenngochonghoa/Excel-US-Superstore-Business-Analytics-Dashboard)

This project presents an interactive Excel dashboard analyzing retail sales and profit performance using the US Superstore dataset (2014–2017). Using Pivot Tables, slicers, and VBA automation, I built a structured reporting system to track key KPIs across categories, regions, and customer segments.
The analysis reveals strong sales growth in 2017 alongside declining profit margins. High-performing subcategories (Appliances, Phones, Binders) contrast with underperforming ones (Machines, Tables), while regional disparities and Q4 seasonality highlight optimization opportunities.
Based on these insights, I proposed strategies to improve product mix, pricing, and seasonal sales planning. This project demonstrates my ability to apply Excel analytics and deliver actionable business insights.

### Nashville Housing Data Cleaning  
**Tools:** SQL  
🔗 [View Project](https://github.com/nguyenngochonghoa/Nashville-Housing-Data-Cleaning-Analysis-in-SQL)

This project focuses on cleaning and analyzing Nashville housing data to uncover real estate trends and investment opportunities. Using SQL (T-SQL), I transformed raw data into a structured dataset by handling missing values, standardizing formats, and removing duplicates.
The analysis highlights strong market growth between 2013–2016, higher property values in luxury suburbs, and the impact of housing features on pricing, with bathrooms showing stronger influence than bedrooms. Undervalued properties and seasonal patterns were also identified, supporting investment decision-making.
Based on these insights, recommendations were proposed to optimize investment timing, target high-potential segments, and improve pricing strategies. This project demonstrates my ability to combine data cleaning, SQL analysis, and business thinking to generate actionable insights.
